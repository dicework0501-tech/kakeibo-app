# よくある質問

## ・夫婦でスマホでオンラインで使えるようにすると伝えたけど、PCでも更新して同期できるようにできるかな？

**できます。** そのままで大丈夫です。

- このアプリは **Webアプリ** なので、デプロイした **同じURL** を、**PCのブラウザ** でも **スマホのブラウザ** でも開けます。
- **クラウド同期（Supabase）** を有効にすると、データは「世帯ID」単位でクラウドに保存されます。
- なので、**PCで更新** → クラウドに保存 → **妻のスマホ・夫のスマホ** で開くと同じ内容になります。逆にスマホで更新した内容も、PCで開けば同じように反映されます。
- 端末は「PC」「妻のiPhone」「夫のAndroid」のどれで更新しても、同じデータが共有されます。追加の設定は不要です。

## ・実装後の修正も問題なくできるかな？ここで修正するだけじゃなくてデプロイ？的な作業は発生するかな？

**コードを直したあと、オンラインで使っている場合は「デプロイ」作業が発生します。**

- **ローカル（自分のPC）でだけ使う場合**  
  - コードを直す → `npm run dev` で起動し直せば、**その場で反映**されます。デプロイは不要です。

- **Vercel などにデプロイして、夫婦でオンラインのURLを使っている場合**  
  - コードを直すだけでは、**そのURLで開いている画面には反映されません**。
  - 反映させるには **もう一度デプロイ** する必要があります。
    - 例: GitHub と連携しているなら、修正をコミットして `main` に push すると、Vercel が自動で再デプロイしてくれます。
    - 手動デプロイの場合は、Vercel のダッシュボードから「Redeploy」するか、同じ手順でもう一度デプロイします。
- まとめ: **「ここで修正する」＋「デプロイ的な作業（push や Redeploy）」の両方があると、オンラインで使っている画面に修正が反映されます。**

## ・使うAIは今Geminiにしているけど選択できる仕様にすることはできるかな？

**できます。** 設定画面に「AIプロバイダー」の選択を追加してあります。

- いまは **Gemini** のみ選択可能です。将来、OpenAI（ChatGPT）や Claude などを追加するときは、同じ「プロバイダー選択」の仕組みに乗せて増やしていけます。
- 設定タブで「AIプロバイダー」を選び、選んだプロバイダー用の API キーを入力・保存すれば、AI資産分析とFP診断でそのプロバイダーが使われます（現状は Gemini のみ実装）。
